BA Rate My Posts allows all users to view rating and allows logged in users to rate any post or page the shortcode is inserted into.

== Description ==

BA Rate My Posts provides a way for logged in users to rate any post the shortcode is inserted into.
All users are able to view the rating, but only logged in users with the Read capability can rate.
Admin can choose which icon to to use for ratings across site.
Currently you can choose from: star, heart, carrot, paw, and diamond.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `ba-rate-my-posts.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
